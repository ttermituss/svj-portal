## Project Overview

This is a **document repository** for SVJ Pivovarska 936 (Společenství vlastníků jednotek — Property Owners Association) in Prague. It contains administrative documents, templates, and bylaws for managing the association.

**Website**: https://pivovarska936.svjo.cz/

## Repository Structure

- `source/` — all project files
  - `source/dokumenty/` — document templates (meeting invitations, letter templates, resident contacts spreadsheet)
  - `source/dokumenty/old/` — archived older versions of documents
  - `source/stanovy/` — association statutes/bylaws (organized by year)
  - `source/info.txt` — project notes
  - `source/SVJ_podklad_kontakty_agendy_havarie.docx` — contacts and emergency agenda reference

## Language

All documents are in **Czech**. Communicate in Czech when working with document content.

## Key Context

- SVJ = Společenství vlastníků jednotek (Property Owners Association)
- The building is at Pivovarska 936, Prague
- Documents include meeting invitations, statutes, contact lists, and letter templates
- There is no source code, build system, or test infrastructure — this is purely document management
